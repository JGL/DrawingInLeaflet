Who can edit, and token safety

Opening this page or having a GitHub account does not let you edit the map. Saving needs a token with write access to this repository, and GitHub only issues such a token to the repo owner or a collaborator with write access (here, Aleks). A token made by anyone else is scoped to their own repos and is rejected when it tries to save here.

To get a token: GitHub → Settings → Developer settings → Fine-grained personal access tokens, scoped to only this repository with Contents: Read and write, and an expiry. It is stored only in this browser (localStorage) and sent solely to api.github.com — never written into the page or the URL. "Remember token" keeps it on this device; "Forget token" clears it. If it ever leaks, revoke it on GitHub; its only power is editing this one repo.
