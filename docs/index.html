Leaflet Drawing Experiments

Please see the course GitHub repo and wiki for other information, including information on reading lists, weekly schedule, assessment details and course outcomes.

Experiments:

  1. Leaflet boilerplate.
  2. Custom Drawing.
  3. Save out GeoJSON by pressing capital S.
  4. Load GeoJSON by using the button at the bottom of the page.
  5. Load GeoJSON that has titles for polygons and multimedia links in popups.

If you want to see some custom GeoJSON with multimedia already added, which you could load into Load GeoJSON that has titles for polygons and multimedia links in popups, then save this file (tourWithTwoPolygonsAndLabelsAndMultimedia.geojson) to your desktop or somewhere else local and upload it to the previously mentioned example - each polygon will have a label which is a weblink to an image - best to right click the link and open in another tab or window so that you don't lose the data you've already loaded!

Please see the GitHub repo of this project for source code and the like.
