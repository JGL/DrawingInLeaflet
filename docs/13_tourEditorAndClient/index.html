Holland Park Tour — Editor & Client

A static, GeoJSON-driven tour builder. The editor creates locations (centre + radius geofences) with a name, text and an image; the client plays the tour back, showing media automatically as a visitor walks into each zone.

  • Editor — draw circular zones, attach name / text / image, import and export tour.geojson.
  • Client — read-only playback with live geolocation, honest GPS accuracy circle and automatic multimedia.

Workflow: build a tour in the editor, Export GeoJSON, save it as data/tour.geojson in this folder (and commit any new images to img/), then open the client.
